# AI Financial Report Analyzer

## Overview

AI-powered application that analyzes financial statement images and generates concise financial summaries.

The system extracts important financial information from balance sheets, annual reports, earnings reports, charts, and graphs to help investors and analysts make informed decisions quickly.

---

## Features

- Financial document image analysis
- Extraction of key financial metrics
- Balance sheet summarization
- Trend identification
- Risk assessment
- Graph and chart interpretation
- AI-generated financial insights

---

## Technologies Used

- Python
- Google Gemini API
- PIL (Python Imaging Library)
- Financial Data Analysis
- Prompt Engineering

---

## Project Workflow

1. Upload financial statement images
2. Process images using PIL
3. Send images to Gemini AI
4. Extract financial information
5. Generate structured financial summary

---

## Output Includes

- Revenue Analysis
- Net Profit Analysis
- Earnings Per Share (EPS)
- Growth Trends
- Balance Sheet Highlights
- Risk Factors
- Management Outlook
- Financial Ratios
